基于小程序云开发的旅游团建小程序源码。该小程序前端采用小程序，后端用云开发，后台用的是cms系统。功能介绍如下：
小程序端：

1.轮播图管理
2.景区展示，
3.景区搜索，
4.景区具体信息浏览
5.景区推荐
6.路线推荐
7.路线推荐详情
8.跟团游方案管理
9.定制出行景区方案管理
10.下单管理
后台：
1.景区管理
2.路线方案管理
3.订单管理
4.等
